## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cekresi'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cekresi

## Usage

Just simple
```ruby
Cekresi.fetch("your tracking_number")
```
